**转义字符（Escape Character）** 是一种特殊的字符常量。以反斜杠 **`\`** 开头，后跟一个或多个字符，用来表示那些无法直接打印的控制符号（如换行、退格）或具有特殊语法的字符（如引号、反斜杠本身）。



#### 简单的转义字符示例

```c
printf("abcdef\n");  //换行
```



#### 字符串的结束

```c
printf("abcd\0ef");   //打印abcd
```



### 常用转义字符对照表



| **转义字符** | **名称**               | **作用与含义**                                               |
| ------------ | ---------------------- | ------------------------------------------------------------ |
| **`\n`**     | 换行 (Newline)         | 将光标移动到下一行的开头（最常用）。                         |
| **`\t`**     | 水平制表符 (Tab)       | 将光标向右移动到下一个制表位（通常是8个空格的位置），用于对齐。 |
| **`\0`**     | 空字符 (Null)          | ASCII值为0，**用于标识字符串的结束**。                       |
| **`\\`**     | 反斜杠                 | 在字符串中输出一个真实的反斜杠 `\`。                         |
| **`\'`**     | 单引号                 | 用于在字符常量中表示单引号（如 `char c = '\'';`）。          |
| **`\"`**     | 双引号                 | 用于在字符串中表示双引号，避免被误认为是字符串的结尾。       |
| **`\b`**     | 退格 (Backspace)       | 将光标向左回退一格（不删除字符，但在后续输出时会覆盖）。     |
| **`\r`**     | 回车 (Carriage Return) | 将光标移动到**当前行**的开头（常用于覆盖当前行输出）。       |
| **`\a`**     | 警报 (Alert)           | 触发系统蜂鸣器响铃或屏幕闪烁。                               |



### 数值转义字符

可以通过字符的 ASCII 码值（八进制或十六进制）来直接表示任意字符。

- **`\ddd`**：`ddd` 表示 1 到 3 位八进制数。
  - *例子：* `\101` 代表 ASCII 码为 65 的字符（即大写字母 'A'）。
- **`\xhh`**：`hh` 表示 1 到 2 位十六进制数（`x` 必须小写）。
  - *例子：* `\x41` 也代表 ASCII 码为 65 的字符（即大写字母 'A'）。

### 核心注意事项

> **存储占用：** 无论转义字符看起来由几个符号组成（例如 `\n` 是两个符号，`\101` 是四个符号），在内存中它们**只占 1 个字节**（即被视为 1 个字符）。

### 转义字符的用法

在字符串中打印真实的`\0`：

```c
printf("abcde\\0f");  //输出包含\0
```
