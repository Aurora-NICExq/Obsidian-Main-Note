### 逻辑操作符类型
```c
&&     //逻辑与
||     //逻辑或
```
- 区分逻辑与和按位与
- 区分逻辑或和按位或

逻辑运算的对象是真假，非零为真，零为假
逻辑运算的结果：
- 真为1
- 假为0

#### 闰年判断逻辑示例
```c
int main(){
	int year;
	if((year%4==0) && (year%100!=0) || year%400==0);
}
```


#### 短路求值
`&&`逻辑短路
```c
int i=0,a=0,b=2,c=3,d=4;  
i = a++ && ++b && c++ && d++;   //a++后置自增，需要先取值
printf(" a=%d b=%d c=%d d=%d\n",a,b,c,d);
```
**一旦 `&&` 左边的表达式结果为“假”（0），系统就会立即停止向右执行，因为整个表达式的结果已经注定是“假”了。**

`||`逻辑短路
```c
int i=0,a=1,b=2,c=3,d=4;
i = a++ || ++b || d++;
```
**一旦 `||` 左边的表达式结果为“真”（1），系统就会立即停止向右执行，因为整个表达式的结果已经注定是“真”了。**

## 条件操作符
```C
exp1 ? exp2 : exp3
```

#### 简单示例
if语句下的逻辑
```c
int a=3,b-0;
if (a>5){
	b=3;
}
else b=-3;
```
使用条件操作符:
```c
(a>3) ? (b=3) : (b=-3)
b= (a>5)? 3 : -3;  //另一种表达方式
```

#### 条件操作符的作用
实现两个数中寻找最大值

## 逗号表达式
```c
exp1,exp2,exp3,...epxn;
```
（用逗号隔开的多个表达式）
**从左向右依次执行，整个表达式的结果是最后一个表达式的结果**
[[编程语言/c语言/c语言基础/c语言入门/操作符/练习题#^1a61f0|习题]]
#### 示例
```c
int a=1, b=2;  
int c=(a>b,a=b+10,b=a+1);  
printf("%d\n",c);    //输出结果为13
```
