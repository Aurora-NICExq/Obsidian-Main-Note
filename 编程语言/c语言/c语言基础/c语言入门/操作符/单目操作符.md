#### `sizeof`

^948eb7

**`sizeof`操作数的类型长度（以字节为单位）**
```c
int a=0;
int n=sizeof a;    //sizeof不是函数，可以不加括号
int m=sizeof(a);   //两种写法等价
printf("%d,%d",m,n);  
```

作用：
- 计算变量占据空间大小
- 计算类型所创建的变量占据空间大小
#### `~`操作符
`~`对数据的补码进行按位取反。[[算术操作符#^60382d|数据的二进制表示]]
```c
int a=0;
printf("%d\n",~a);  //输出结果是-1
```

`~` 运算符的数学捷径公式：

 **`~x = -(x + 1)`**

#### `--`和`++`操作符
##### 前置运算
```c
int a=0;
int b=++a;         //先++，后使用
printf("%d\n",b);  //输出b为1
```

##### 后置运算
```c
int a=0;
int b=a++;          //先使用，后++
printf("%d\n",b);   //输出为0
```

**`--`和`++`运算符作用方式相同**
