## 下标引用操作符
1. `[]`下标引用操作符
操作数：一个数组名+一个索引值
```c
int arr[10]; //创建数组
arr[9]=10;   //使用下标引用操作符
```
`[]`的两个操作数是`arr`和`9`

#### 示例
 ```c
 arr[7]=9;
 7[arr]=9;   //两种写法等价
 ```

## 函数调用操作符
1. `()`函数调用操作符
操作数：函数名+参数
```c
int c=Add();  //()是操作符
```

## 结构成员操作符
1.   `.`结构体.成员名
2.   `->`结构体指针->成员名

```c
struct student{
	int age;
}
struct student lww;
printf lww.age  //用.访问
```
