指针类型为字符指针`char*`

## 字符指针的用法
一般用法：
```c
char ch='t';
char* pi=&ch;
*pi='a';
```

字符串指针：
```c
char* p="abcdef";  //把字符串首字符a的地址赋给p
printf("%s\n",p);  //不能用*p
```

#### 错误示例
```c
char ch="abcde";  //常量字符串
char* p=&ch;
*p="w";           //无法强行修改
```


#### 字符串数组和字符串指针区别
```c
const char* p1="abcdef";
const char* p2="abcdef";   //p1==p2

char arr1[]="abcdef";
char arr2[]="abcdef";      //arr1!=arr2
```

##### 原因
`p1`和`p2`指针都指向不能更改的常量字符串
