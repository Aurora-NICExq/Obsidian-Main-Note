## 基本结构

双向链表的每个结点通常包含三部分：

- **数据域**：存你要管理的数据
- **前驱指针 prev**：指向前一个结点
- **后继指针 next**：指向后一个结点

### 代码实现

```c
typedef int ElemType;
typedef struct Dnode{
  ElemType data;
  struct Dode * prev,*next;  //*prev表达前驱
}DNode;
```

### 双链表的优势

单向链表的缺点：

> 如果只有某个结点 `p`，想删除它——必须先找到它的前驱结点，才能把前驱的 `next` 接过去。
>  这会导致“删除/插入前面”经常不得不从头遍历。

双向链表多了 `prev`，于是：

- **已知结点指针 p 时，删除 p 可以 O(1)**（不需要找前驱）
- **从尾到头逆向遍历**很自然

## 双链表插入元素

### 头插法



#### 代码实现

```c
int insertHead(Node* L,ElemTypre e)  //只写了指针相关代码
{
  p->data = e;
  p->prev = L;
  p->next = L->next;
  if (L->next != Null)
  L->next->prev = p;
}
L->next = p;
```
