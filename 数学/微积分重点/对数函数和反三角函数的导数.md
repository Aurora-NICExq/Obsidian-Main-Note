## 对数函数的导数

反函数法证明：	

$$(\ln x)' = \frac{1}{x}$$



### 证明步骤

1. 令 $y = \ln x$。

2. 根据对数的定义，这等价于 $x = e^y$。

3. 现在，我们在等式 $x = e^y$ 两边同时对 $x$ 求导。

   - 左边 $x$ 的导数是 $1$。
   - 右边 $e^y$ 是一个复合函数（$y$ 是 $x$ 的函数），根据**链式法则**，它的导数是 $e^y \cdot y'$。

4. 于是得到方程：

   

   $$1 = e^y \cdot y'$$

5. 解出 $y'$：

   

   $$y' = \frac{1}{e^y}$$

6. 因为最开始我们知道 $x = e^y$，直接代入：

   

   $$y' = \frac{1}{x}$$



## 反三角函数的导数



### $\arcsin x$ 的导数

证明目标公式：



$$(\arcsin x)' = \frac{1}{\sqrt{1-x^2}}$$

**推导步骤：**

第一步：（转化为普通三角函数）

令 $y = \arcsin x$。

根据反函数的定义，这意味着：



$$x = \sin y$$



(注意： $y$ 的范围被限制在 $[-\frac{\pi}{2}, \frac{\pi}{2}]$，这个范围内 $\cos y$ 总是正的)

第二步：两边求导（隐函数求导）

方程两边同时对 $x$ 求导：

- 左边 $x$ 的导数是 $1$。
- 右边 $\sin y$ 是复合函数，导数是 $\cos y \cdot y'$。

得到：



$$1 = \cos y \cdot y'$$

第三步：解出 $y'$



$$y' = \frac{1}{\cos y}$$

第四步：回代（把 $y$ 变回 $x$）

现在的导数是用 $y$ 表示的，把它变回用 $x$ 表示。

要利用三角恒等式 $\sin^2 y + \cos^2 y = 1$。

因为 $x = \sin y$，所以：



$$\cos y = \sqrt{1 - \sin^2 y} = \sqrt{1 - x^2}$$

最终得到：



$$(\arcsin x)' = \frac{1}{\sqrt{1-x^2}}$$

### $\arctan x$ 的导数

证明：

$$(\arctan x)' = \frac{1}{1+x^2}$$

**推导步骤：**

令 $y = \arctan x$，即：

$$x = \tan y$$

第二步：两边求导

两边对 $x$ 求导。

- 回忆一下：$(\tan y)' = \sec^2 y$ （即 $\frac{1}{\cos^2 y}$）。

- 利用链式法则：

  

  $$1 = \sec^2 y \cdot y'$$

第三步：解出 $y'$

$$y' = \frac{1}{\sec^2 y}$$

第四步：回代

三角恒等式：$1 + \tan^2 y = \sec^2 y$。

因为 $x = \tan y$，所以：

$$\sec^2 y = 1 + x^2$$

直接代入：

$$(\arctan x)' = \frac{1}{1+x^2}$$



## 总结与规律

| **函数 f(x)**   | **导数 f′(x)**               | **备注**                  |
| --------------- | ---------------------------- | ------------------------- |
| **$\arcsin x$** | **$\frac{1}{\sqrt{1-x^2}}$** | **核心公式，必须掌握**    |
| $\arccos x$     | $-\frac{1}{\sqrt{1-x^2}}$    | 与 $\arcsin$ 仅差一个负号 |
| **$\arctan x$** | **$\frac{1}{1+x^2}$**        | **核心公式，没有根号**    |
| $arccot x$      | $-\frac{1}{1+x^2}$           | 与 $\arctan$ 仅差一个负号 |
| $arcsec x$      |                              | x                         |

- **正割的反函数**：$(\text{arcsec } x)' = \frac{1}{|x|\sqrt{x^2-1}}$
- **余割的反函数**：$(\text{arccsc } x)' = - \frac{1}{|x|\sqrt{x^2-1}}$
